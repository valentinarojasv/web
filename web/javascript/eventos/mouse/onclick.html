Onclick

—Este atributo responde al evento click. El evento se ejecuta cuando el usuario hace clic con el botón izquierdo del ratón. HTML ofrece otros dos atributos similares llamados ondblclick (el usuario hace doble clic con el botón izquierdo del ratón) y oncontextmenu (el usuario hace clic con el botón derecho del ratón).
